# KELYST EXTENSION

## REQUIREMENTS
For this project you need to use
- node v14.3.0
- npm 6.9.0
- yarn 1.12.1

## HOW TO INSTALL THE DEPENDENCIES
```
$ npm install

// or

$ yarn
```

## HOW TO BUILD
```
$ npm run build

// or

$ yarn build
```
> The code generated by webpack is now in the ./build folder, you can contact my at any time with the live messenger at www.kelyst.com
